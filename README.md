# CPSC 4860: The Chthonic Kernel

**The Chthonic Kernel** is a text-based RPG set in a mythic-academic underworld, blending Greek mythology with Computer Science culture.

## Play the Game
[**Click here to Play**](https://awibisono.github.io/cpsc4860/) (Once hosted on GitHub Pages)

## Documentation
*   [**Design Document (The Cave)**](cave.html): Detailed lore, mechanics, stats, and script.

## Premise
You are a Neophyte Scholar who has segfaulted into the **Underworld Kernel**. Guided by **Cerberus Dan**, you must traverse the corrupted Colleges of the Dead, defeat the **Archons**, and compile the *Lux et Veritas* to graduate with your soul intact.

## Features
*   **Hybrid Terminal Interface:** Playable on Desktop (Keyboard) and Mobile (Touch controls).
*   **Responsive Design:** Adapts to any screen size.
*   **Four Elemental Kingdoms:** Sky, Sea, Fire, Spirit.
*   **Boss Battles:** Debate and battle corrupted Professors.

## Tech Stack
*   Single-file HTML/JS engine.
*   No external dependencies.
